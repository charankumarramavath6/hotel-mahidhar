1. Install Node.js

Download and install Node.js from https://nodejs.org/

node --version # Verify installation

2. Install MySQL

Download and install MySQL from https://dev.mysql.com/downloads/mysql/

Or use XAMPP for easier setup: https://www.apachefriends.org/

3. Install Dependencies

npm install

4. Configure Database

Edit server.js and update MySQL connection:

const dbConfig = { host: 'localhost', user: 'root', password: 'your_mysql_password', // Change this database: 'hotel_mahi', waitForConnections: true, connectionLimit: 10, queueLimit: 0 };

5. Start the Server

npm start

Then open: http://localhost:3000
